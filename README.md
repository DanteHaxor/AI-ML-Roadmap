# AI/ML Roadmap – Beginner to Advanced 🚀

This repository contains a **complete AI/ML learning roadmap** using Python.
It is designed for beginners as well as intermediate learners who want hands-on practice.

## 📌 Topics Covered
- Python for Data Science
- NumPy
- Pandas
- Data Collection (APIs & Web Scraping)
- SQL for Data Science
- Data Visualization
- Math for Machine Learning
- Machine Learning (Beginner → Advanced)

## 📂 Repository Structure
Follow folders in order:
00_Python_Basics → 01_NumPy → 02_Pandas → 03_Data_Collection → 04_SQL →
05_Data_Visualization → 06_Math_for_ML → 07_Machine_Learning


## 🧠 How to Use
1. Clone the repository
2. Install required libraries
3. Run notebooks in sequence
4. Practice and modify the code
