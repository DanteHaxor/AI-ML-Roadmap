# AI/ML Roadmap – Beginner to Advanced 🚀

This repository contains a **complete AI/ML learning roadmap** using Python.
It is designed for beginners as well as intermediate learners who want
**strong fundamentals, hands-on practice, and industry-ready skills**.

The roadmap follows a **step-by-step progression**, starting from Python basics
and moving toward Machine Learning, Deep Learning, and Generative AI.

---

## 📌 Topics Covered

### ✅ Core Foundations
- Python for Data Science  
- NumPy  
- Pandas  
- Data Collection & Preprocessing
- SQL for Data Science  
- Data Visualization  
- Mathematics for Machine Learning  

### ✅ Machine Learning
- Supervised Learning  
- Unsupervised Learning  
- Reinforcement Learning  

### ✅ Deep Learning & Generative AI
- Neural Networks  
- Perceptron  
- Feedforward Neural Networks (FNN)  
- Recurrent Neural Networks (RNN)  
- Convolutional Neural Networks (CNN)  
- Transformer Architecture  
- PyTorch  
- Generative AI  
- Large Language Models (LLMs)  
- Natural Language Processing (NLP)  
- Retrieval-Augmented Generation (RAG)  
- Generative Adversarial Networks (GANs)  
- Agentic AI

## 📂 Repository Structure
Follow folders in order:
00_Python_Basics → 01_NumPy → 02_Pandas → 03_Data_Collection → 04_SQL →
05_Data_Visualization → 06_Math_for_ML → 07_Machine_Learning → (Deep Learning & GenAI folders will be added)

Each folder builds upon the previous one and should be completed **in order**.

---

## 🧠 How to Use This Repository

1. Clone the repository  
2. Install required Python libraries  
3. Run notebooks sequentially  
4. Experiment with code and datasets  
5. Modify notebooks to build your own understanding  

This repository is meant to be **learned by doing**, not just reading.
